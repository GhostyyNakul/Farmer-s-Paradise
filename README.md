# Farmer-s-Paradise
The Farmer's PAradise is an amazing solution for the issues faced by the farmers around the globe. It uses Mysql for storing the data and further allows farmer to know abt its soil details and crops to grow on it. 
it uses different tables in the sql where the serves is named as "system 24" and the tables are as so : "soil", "crops", "disease", "solution", "farmer". 
Further it uses a lot of libraries in pythoon to make the user interface interactive including tkinter, messagebox and connectivity of python with sql database.
